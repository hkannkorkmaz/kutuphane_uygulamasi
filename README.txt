# Kütüphane Uygulaması

Bu proje, **Zion** tarafından geliştirilen bir Python tabanlı kütüphane yönetim uygulamasıdır. Uygulama kitap verilerini yönetmek, barkodları okumak ve görselleri organize etmek için geliştirilmiştir.

## Özellikler
- Kitap verilerini kaydetme ve düzenleme
- Barkod okutma desteği
- Kapak görsellerini organize etme
- Hafif ve hızlı Python uygulaması

## Gereksinimler
- Python 3.13 veya üstü
- Pyzbar kütüphanesi (barkod okuma için)
- OpenCV (gerekirse görüntü işleme)

## Kurulum
Bu repo **kaynak kodları ve örnek veri dosyalarını** içerir.  

Eğer hazır bir **exe veya setup dosyası** istiyorsanız, lütfen benimle iletişime geçin; size özel olarak gönderebilirim.

Kaynak koddan çalıştırmak için:

1. Depoyu klonlayın veya ZIP olarak indirin.
2. Gerekli paketleri yükleyin:

```bash
pip install pyzbar opencv-python

app.py dosyasını çalıştırın: python app.py

Notlar

dist veya build klasörleri ve exe/setup dosyaları GitHub’da bulunmamaktadır.
